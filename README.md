# 🐱 고양이를 쓰다듬는다

## 1. 목적
- 고양이를 쓰다듬는다.
- 고양이 이외의 대상도 쓰다듬을 수 있다.
- 궁극적으로 모든 것을 쓰다듬는다.

## 2. 사용법
1. **쓰다듬기 버튼**을 눌러 기본 고양이를 쓰다듬는다.
2. **고양이 추가 버튼**을 눌러 원하는 고양이 사진을 선택한다.
3. 선택 후  
   - **PC:** 우측 선반에서 쓰다듬을 고양이 사진 선택  
   - **모바일:** 상단 선반에서 쓰다듬을 고양이 사진 선택
4. 이후 다시 **쓰다듬기 버튼**을 눌러 해당 고양이를 쓰다듬는다.

## 3. 사용 기술
- ⚛️ React  
- 🟦 TypeScript
